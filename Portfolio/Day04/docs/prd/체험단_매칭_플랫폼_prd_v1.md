## 1. 제품 개요

### 배경 & 목적
포트폴리오용 체험단 매칭 플랫폼으로, 광고주와 인플루언서를 연결하여 제품/서비스 체험단을 운영할 수 있는 중개 플랫폼

### 타겟 사용자
- **광고주**: 제품/서비스 체험단을 모집하려는 업체
- **인플루언서**: 체험단에 참여하여 SNS에 후기를 작성하는 개인

### 성공 지표
| 지표 | 베이스라인 | 목표 | 측정 방법 |
|------|------------|------|-----------|
| 페이지 구현 완료율 | 0% | 100% | 수동 체크리스트 |
| 회원가입 플로우 완료율 | 0% | 100% | 테스트 시나리오 |
| 권한 제어 정확도 | 0% | 100% | 기능 테스트 |

### 범위
- ✅ In Scope: 9개 핵심 페이지, 회원가입/로그인, 역할별 정보 입력, 체험단 CRUD, 권한 제어
- ❌ Out of Scope: 결제 시스템, 실시간 채팅, 모바일 앱, 알림 기능

## 3. 사용자 여정 (주요 시나리오)

### 광고주 여정
1. 홈페이지 접속 → 2. 회원가입 → 3. 광고주 선택 → 4. 업체 정보 입력 → 5. 체험단 등록 → 6. 대시보드에서 관리

### 인플루언서 여정
1. 홈페이지 접속 → 2. 회원가입 → 3. 인플루언서 선택 → 4. SNS 정보 입력 → 5. 체험단 탐색 → 6. 지원

## 4. 페이지 구성

### A. 홈 - 체험단 탐색
- **목적**: 체험단 목록 표시 및 플랫폼 소개
- **접근 권한**: 모든 사용자 (비로그인 포함)
- **주요 기능**: 체험단 목록, 검색, 로그인/회원가입 링크

### B. 로그인
- **목적**: 기존 사용자 인증
- **접근 권한**: 비로그인 사용자
- **주요 기능**: 이메일/비밀번호 로그인, 회원가입 링크

### C. 회원가입
- **목적**: 신규 사용자 등록
- **접근 권한**: 비로그인 사용자
- **주요 기능**: 기본 정보 입력, 역할 선택으로 이동

### D. 광고주 정보 등록
- **목적**: 광고주 전용 정보 입력
- **접근 권한**: 역할 선택에서 광고주 선택한 사용자
- **주요 기능**: 업체명, 주소, 업장 전화번호, 사업자등록번호, 대표자명

### E. 인플루언서 정보 등록
- **목적**: 인플루언서 전용 정보 입력
- **접근 권한**: 역할 선택에서 인플루언서 선택한 사용자
- **주요 기능**: SNS 채널명, 채널링크, 팔로워수, 생년월일

### F. 체험단 상세
- **목적**: 개별 체험단 상세 정보 표시
- **접근 권한**: 모든 로그인 사용자
- **주요 기능**: 상세 정보, 지원하기 버튼(인플루언서만)

### G. 체험단 지원
- **목적**: 체험단 지원 신청
- **접근 권한**: 인플루언서 정보 등록 완료한 사용자만
- **주요 기능**: 지원서 작성, 제출

### H. (광고주) 체험단 관리 대시보드
- **목적**: 광고주의 체험단 관리 허브
- **접근 권한**: 광고주 정보 등록 완료한 사용자만
- **주요 기능**: 체험단 목록, 등록, 수정, 삭제

### I. (광고주) 체험단 상세
- **목적**: 광고주용 체험단 상세 관리
- **접근 권한**: 광고주 정보 등록 완료한 사용자만 (본인 체험단만)
- **주요 기능**: 지원자 목록, 모집 조기 종료, 인플루언서 선정

## 5. 비즈니스 규칙

### 회원가입 & 역할
- 회원가입 직후 반드시 광고주/인플루언서 역할 선택
- 공통 정보: 이름, 생년월일, 휴대폰번호
- 역할별 추가 정보 입력 필수

### 접근 제어
- 인플루언서 정보 등록 완료자만 체험단 지원 가능
- 광고주 정보 등록 완료자만 광고주 대시보드 접근 가능
- 체험단 관리는 해당 체험단 등록자만 가능

### 체험단 운영
- 광고주는 모집을 조기 종료 가능
- 모집 종료된 체험단에서만 인플루언서 선정 가능
- 선정된 인플루언서는 체험단 진행 상태로 변경

## 6. 기술 요구사항

### 성능
- 페이지 로드 시간 < 3초
- 이미지 최적화 (WebP, 압축)

- 비밀번호 해싱 (bcrypt)
- 세션 관리
- CSRF 보호
- 입력 값 검증

### 호환성
- 모던 브라우저 지원 (Chrome, Firefox, Safari, Edge 최신 2버전)
- 반응형 디자인 (데스크톱 우선)
