## 개발 가이드

### 초기 환경 설정

```bash
# 1. Vite 프로젝트 생성
npm create vite@latest obstacle-game -- --template vanilla

# 2. 의존성 설치
cd obstacle-game
npm install

# 3. 개발 서버 실행
npm run dev

# 4. Git 초기화
git init
git add .
git commit -m "Initial commit"
```

### 배포 설정

```bash
# 1. GitHub에 저장소 생성 후
git remote add origin <your-repo-url>
git push -u origin main

# 2. Vercel 연결
# https://vercel.com에서 GitHub 저장소 연결
# 자동 배포 완료!
```

### 권장 파일 구조

```
obstacle-game/
├── index.html          # 메인 HTML
├── src/
│   ├── main.js         # 진입점
│   ├── game.js         # 게임 로직
│   ├── player.js       # 플레이어 클래스
│   ├── obstacle.js     # 장애물 클래스
│   └── utils.js        # 유틸리티 함수
├── style.css           # 스타일
├── README.md           # 프로젝트 설명
└── package.json        # 의존성 관리
```
